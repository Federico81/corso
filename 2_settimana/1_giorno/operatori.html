Operatori

Tramite gli operatori posso combinare valori tra di loro

Gli operatori determinano il risultato di un'operazione

Classificazione degli operatori in base al numero di valori che possono combinare: unari, binari, ternario

Operatori aritmetici

operatori binari perché combinano due valori (numerici!)
+ addizione
- sottrazione
/ divisione
* moltiplicazione
% modulo o resto

operatori aritmetici possono anche essere unari, cioè combinano un solo valore
++ incremento
-- decremento
- negazione -> x=5 -> -x = -5
se ci sono due valore - è sottrazione, se c'è un valore solo - è negazione

-> incremento e decremento li posso applicare solo a delle variabili
-> è molto importante la posizione dell'operatore (se a sinistra oppure a destra dell'operando)
-> se l'operatore è a sinistra dell'operando l'operazione viene eseguita prima di valutare l'espressione
-> se l'operatore è a destra dell'operando l'operazione viene eseguita dopo

Operatori logici

Consentono di combinare espressioni booleane (true | false)
&& -> operatore binario -> and
|| -> operatore binario -> or
! -> operatore unario -> not

Operatore ternario

Permette di combinare tre dati tra di loro. Restituisce un valore in base alla valutazione di un'espressione booleana
Sintassi: condizione? valore1 : valore2
Verifico se la condizione è vera (true|false) -> se è true restituisce valore1 altrimenti valore2

Operatori relazioni

utilizzati per confrontare dei valori

OperatoreNome
> maggiore
< minore
>= maggiore o uguale
<= minore o uguale
== uguale
!= diverso
=== strettamente uguale
!== strettamnete diverso

tutte queste espressione restituiscono un valore booleano (true|false)

Operatori su stringhe

Operatore concatenazione (+) -> crea una nuova stringa come risultato della concatenazione di due stringhe
