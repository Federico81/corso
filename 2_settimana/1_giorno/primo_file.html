Inserisco blocchi di codice javascript

Pro: nel caso in cui abbiamo poco codice javascript lo metto subito in relazione con i tag html con cui deve interagire

Contro: nel caso - comune - in cui il codice js sia minimamento complesso (variabili, funzioni, oggetti) non risultato comodo.

Tag head o tag body? primo ordine di considerazioni: performance
secondo ordine di considerazioni -> che il tag html su cui js interagisce sia già stato parsato (letto e quindi disponibile) -> questo spiega perché tante volte troviamo gli script in fondo alla pagina prima della chiusura di </body>

La console è uno strumento fondamentale per scrivere codice javascript (in qualsiasi linguaggio di programmazione)

La console ci permette di fare il debug cioè di controllare il flusso del nostro codice

Spesso la console si usa per tenere traccia dell'evoluzione del programma o per verificare valori e trovare errori
