Tipi di dati

Javascript prevede cinque tipi di dati primitivi: numeri, stringhe, booleani, null, undefined
e un tipo di dato complesso: gli oggetti

Qualsiasi altro elemento previsto nel linguaggio js (array, funzioni) sono tipi di dati complessi

Negli oggetti noi possiamo inserire delle proprietà (coppie attributo/valore) e/o metodi (funzioni)

Tipo di dato stringa

E' una sequenza di caratteri delimitata da apici "" oppure '', non c'è regola su che delimitatore usare, unica regola: delimitatore iniziale e finale devono essere uguali. "Questo è una 'stringa'", esiste anche la stringa vuota ""
Per inserire caratteri speciali nelle stringe -> \
"Ieri pioveva.\nOggi c'è il sole" -> \n inserisce un ritorno a capo
'l\'altro ieri' -> il carattere che segue \ viene considerato carattere normale

Tipo di dato numerico

Senza apici. Esiste un solo tipo di dato per tutto (interi, decimali, negativi)

Javascript è un linguaggio a tipizzazione debole -> non c'è controllo sul tipo di dato

Tipo di dato booleano

Ha solo due valori possibili: true | false

tipo di dato null

Prevede un solo valore possibile: null -> non rientra nei precedenti tipi di dato

tipo di dato undefined

Prevede un solo valore possibile: undefined. Rappresenta un valore che non esiste, non ha assegnato nessun valore (nemmeno null!)
