Le variabili

Servono per memorizzare dei valori

Ogni variabile è identificata da un nome che segue alcune regole:

  • niente spazi, niente trattini -, niente caratteri speciali (anche ? oppure .)
  • deve iniziare con una lettera, sono ammessi anche underscore _ e dollare $
  • sono case sensitive
  • non può iniziare con un numero
  • non deve mai coincidere con le keywords di javascript (if, else, switch, true, false, return)

Dichiarazione implicita delle variabili

Non è obbligatorio dichiarare le variabili, cioè fare un'operazione esplicita di creazione
quando c'è l'identificatore = l'engine crea implicitamente la variabile se non esiste già

Var, let, const

var è la prima keyword creata da javascript per definire una variabile, var è accessibile in qualunque parte dellos cript

let è stata introdotto da EcmaScript 6, è accessibile solo in un determinato ambito (scope) delimitato da un blocco di codice

Un blocco di codice javascript è una serie di istruzioni delimitata da { .... blocco di codice .... }

const è stato introdotto da EcmaScript 6 -> istruisce l'engine che il valore assegnato non è modificabile

Strict mode

E' una opzione di controllo del codice, EcmaScript 5 -> se viene dichiarato lo strict mode javascript viene eseguito usando regole più stringenti -> ad esempio se una variabile non viene dichiarata ci viene mostrato un errore -> se dichiariamo lo strict mode -> regole meno permissive -> stile di programmazione più strutturato
