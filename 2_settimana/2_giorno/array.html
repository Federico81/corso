Array

L'array è una variabile multidimensionale. Dentro a una variabile posso inserire un valore, dentro a un array posso inserire più valori

Dentro a un array incontriamo strutture di dati omogenee

Dentro possiamo conservare qualsiasi tipo di dato (primitivi o oggetti)

Gli array sono elenchi, liste in cui salvare diversi dati

Dentro a un array associamo a un unico nome di variabile più valori

L'uso degli arrau ci semplifica le operazioni nel caso in cui dobbiamo definire più variabili

Proprietà degli array

In javascript esistono una serie di metodi che ci permettono di fare operazioni sugli array, la sintassi per fare questa cosa è nome_array.nome_metodo

Il primo metodo che vediamo è length -> conta quanti elementi ci sono in un array

Inserire/rimuovere elementi in un array

per inserire degli elementi in un array abbiamo un metodo che si chiama push

per rimuovere degli elementi da un array abbiamo metodo che si chiama pop

Altri metodi: unshift aggiungere nuovi elementi all'inizio dell'array; shift rimuove dall'inizio dell'array shift e unsfhit sovrascrivono l'array -> gli indici vengono modificati
