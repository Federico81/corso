Logica condizionale

qualsiasi nostro script è una sequenza di istruzioni vengono eseguite dall'engine javascript
tutte le nostre istruzioni (quello che scriviamo nei nostri) sono un flusso di comandi
l'insieme dei comandi che scriviamo porta la nostra pagina web da uno stato iniziale a uno stato finale, con il flusso dei comandi possiamo far raggiungere un certo risultato
avremo bisogno di istruzioni per stabilire il flusso di esecuzione di un programma

Blocchi codice

Le istruzioni vengono raccolte in blocchi che fungono da singole istruzioni. I blocchi di codice ci permettono di eseguire più istruzioni interamente
tipico della logica condizionale è eseguire dei blocchi di codice in seguito alla valutazione di una condizione
un blocco di codice è racchiuso tra parentesi graffe

La logica condizionale è una categoria di istruzioni che consentono di eseguire blocchi di codice sulla base della verifica di una condizione Lo script "decide" se eseguire un blocco di codice, la decisione è basata sulla verifica di una condizione

Javscript prevede due istruzioni condizionali

  • if
  • switch

Il cosrtutto if prevede tre forme diverse

  • if semplice
  • if con alternativa (if..else)
  • if a cascata (if.. else if.. else)

If semplice

La forma if semplice esegue un blocco di codice solo se una condizione è vera
se la condizione non è vera il blocco di codice non viene eseguito

Schema sintattico del costrutto if semplice

If ... else

In questa forma viene eseguito un blocco di codice oppure un altro blocco di codice in base al valore della condizione

Schema sintattico

If a cascata

Mette a disposizione più alternative di esecizioni e quindi la possibilità di verificare più condizioni
