Dom

Document object model è in realtà dell'oggetto window e rappresenta la struttura html caricata nella mia pagina, ha una struttura a oggetto. Il DOM fornisce una rappresentazione gerarchica di oggetti, che viene spesso chiamata Dom tree (assomiglia a un albero genealogico). Questo albero di oggetti è accessibile via javascript, infatti possiamo fare:

  • leggere gli elementi del DOM
  • modificare la struttura degli elementi (tag)
  • modificare il contenuto degli elementi (tag)
  • modificare lo stile

Il DOM si presenta come un albero con una serie di nodi che sono di diversi tipi

Manipolare il DOM via javascript

  • Selezionare un elemento del DOM
  • Modificare lo "style" di un elemento
  • Modificare le classi di un elemento
  • Modificare il contenuto di un elemento
  • Creare e inserire e rimuovere degli elementi nel DOM

documento La radice del mio albero, normalmente ne esiste soltanto una, ma se uso tag <frame&gr; (es. youtube, google maps) ci sono più nodi di tipo documento in una pagina <html>

attributo un nodo di tipo attributo corrsiponde ad un attributo di un tag html ad esempio l'attributo src di un tag immagine; l'attributo href di un link
