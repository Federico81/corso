Parametri rest

Rest in inglese significa resto|restante. Con Paramentro rest raccogliamo un numero non definito di parametri in un array

Quando dichiariamo una funzione indichiamo il numero di parametri -> quelli che tendenzialmente valorizziamo
può capitare in alcune funzioni di vedere n parametri i parametri delle funzioni vengono inseriti in array arguments
Con i parametri rest ... possiamo racchiudere i parametri di troppo

for ...of

for ...of è una variante del ciclo for che esegue un loop su dei valori che vengono da un oggetto iterabile (array ad esempio)

Sintassi -> dichiaro il singolo elemento della collezione
